Javascript Maths Tools

A set of tools using JSXGraph for use in teaching Maths. The tools are as follows:

Trig Graphs

  • Demonstrate the link between sin and cos functions and a circle
  • Sine Circle Graph
  • Cosine Circle Graph

Taylor Series

  • plots a taylor series for a given function alongside the actual graph
  • Taylor Expansion Demo

Double Pendulum

  • Simulation of a double pendulum with any sets of initial conditions. Using an implementation of 4th order Runge Kutta method to solve Hamiltons equations
  • Double Pendulum Simulation: Unit Length and Mass
  • Double Pendulum Simulation: Any IC
These tools were produced by Robert Soane as part of a third year university project working towards a BSc. There are some strange lacks of convention in the programming of these codes.
